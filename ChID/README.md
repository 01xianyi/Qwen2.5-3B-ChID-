  ## 简介
  ChID 是一个用于完形填空测试的大规模中文成语数据集。 ChID 包含 581K 段落和 729K 空白，涵盖多个领域。在 ChID 中，段落中的成语被替换为空白符号。对于每个空白，提供包括黄金成语在内的候选成语列表作为选择。
